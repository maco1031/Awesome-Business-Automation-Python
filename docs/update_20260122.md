## 🚀 Tool #03: Instagram Auto Liker

指定した「ハッシュタグ」がついた投稿を、自動で次々と「いいね」していくスクリプトです。
「#プログラミング初心者」や「#カフェ巡り」など、自分の属性に合ったタグを指定することで、**見込みフォロワーへの認知活動**を自動化できます。

### 機能
*   ユーザー名・パスワードで自動ログイン
*   指定タグの検索
*   ランダムな待機時間（BAN対策）を入れて「いいね」を実行

### 使い方
```bash
python src/03_instagram_auto_like/auto_like.py --username "user" --password "pass" --hashtag "python"
```
