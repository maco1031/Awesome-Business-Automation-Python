# 【Python自動化】寝ている間に「いいね」周り！Instagram運用を効率化するBotを作ってみた

こんにちは。今回は「Pythonでの業務効率化・自動化」シリーズ第3弾です。
SNS運用、毎日手作業でポチポチしていませんか？
今回は、指定したハッシュタグの投稿に自動で「いいね」をしてくれるツールを作成しました。

## 📦 今回作ったツール
**Instagram Auto Liker**
特定のハッシュタグ（例: `#プログラミング初心者`）が付いた投稿を自動で巡回し、いいねを押してくれます。
Seleniumを使ってブラウザを操作するので、普段自分が見ている画面そのままに動きます。

## 🛠️ 技術的な仕組み
- **言語**: Python
- **ライブラリ**: Selenium (Webブラウザ操作)
- **工夫点**:
    - **ランダム待機**: 機械的な動きに見えないよう、クリックごとの待機時間をランダムに変動させています。
    - **ヘッドレスモード**: 画面を出さずにバックグラウンドで実行も可能です。

## ⚠️ 運用の注意
このツールは学習・研究用として公開しています。実際に運用する場合は、InstagramのAPI制限や規約を十分に確認し、アカウント凍結リスクを考慮して慎重に行ってください。
特に、短時間の大量いいねはスパム認定されやすいので注意が必要です。
