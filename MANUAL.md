# Step 1: アカウント開設マニュアル (土台構築)

不労所得システムを稼働させるための「土台」となるアカウント一覧です。
最短で終わらせて、次のステップに進んでください。

## 1. Pinterest ビジネスアカウント (必須)
**「動画」で視覚的に集客するための罠です。**
- **URL**: [https://business.pinterest.com/](https://business.pinterest.com/)
- **設定**:
  - アカウント名: 「Python業務効率化ラボ」など、何が得られるかわかる名前に。
  - プロフィール: 「月200時間削減するPythonコード配布中」 + GitHubリンクを貼る。

## 2. Note / Zenn (必須)
**「テキスト」で検索流入を稼ぐための母艦です。**
- **URL (Note)**: [https://note.com/](https://note.com/) (一般向け)
- **URL (Zenn)**: [https://zenn.dev/](https://zenn.dev/) (エンジニア向け)
- **設定**:
  - 1記事だけ投稿する（内容は `docs/content_draft.md` をコピペ）。
  - **重要**: 記事の最後に必ずアフィリエイトリンクを貼る。

## 3. ASP (アフィリエイト)
**マネタイズの命綱です。**
既にお持ちとのことですが、特に「ConoHa VPS」「Xserver VPS」の提携承認が降りているか確認してください。
- リンク切れは機会損失です。必ずクリックして確認してください。
