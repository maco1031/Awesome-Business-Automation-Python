# Instagram Auto Liker

指定したハッシュタグの人気投稿を巡回し、自動で「いいね」を行うPythonスクリプトです。
Seleniumを使用し、ブラウザを自動操作します。

## ⚠️ 注意事項 (Disclaimer)
- **自己責任でご利用ください**: Instagramの利用規約に抵触する可能性があります。アカウントの制限・凍結リスクを理解した上で使用してください。
- **過度な使用は厳禁**: 短時間に大量のいいねを行うと、スパム判定を受けるリスクが高まります。
- **推奨環境**: 自宅PCではなく、IPアドレスが固定されたVPS等での運用を推奨します。

## 必要要件
- Python 3.x
- Chrome ブラウザ
- Google Chrome (最新版)

## インストール
```bash
pip install -r requirements.txt
```

## 使い方
以下のコマンドを実行します。

```bash
python auto_like.py --username "あなたのユーザー名" --password "あなたのパスワード" --hashtag "プログラミング" --count 20
```

### オプション
- `--username`: Instagramのユーザー名 (必須)
- `--password`: Instagramのパスワード (必須)
- `--hashtag`: 検索対象のハッシュタグ (デフォルト: programming)
- `--count`: いいねする件数 (デフォルト: 10)
- `--headless`: ブラウザ画面を表示せずに実行 (ヘッドレスモード)

## 仕組み
1. Instagramのログインページにアクセスし、自動ログインします。
2. 指定したハッシュタグの検索ページへ移動します。
3. 最新/人気投稿を開き、順番に「いいね」ボタンを押下(シミュレーション)します。
4. ランダムな待機時間を挟むことで、Bot判定を回避しようと試みます。
